Sophia E. Dawson Website Starter

This ZIP is structured for GitHub Pages using the /docs folder on the main branch.

Files included:
- index.html
- reels.html
- gallery.html
- theater.html
- film.html
- commercials.html
- poetry.html
- photography.html
- production.html
- short-stories.html
- contact.html
- styles.css
- script.js

How to use:
1. Replace the files in your repo's /docs folder with these files.
2. Add your real images and update placeholder sections.
3. If you want local images, create an images folder inside /docs and reference them like images/headshot.jpg.
4. For PDF resumes, create a files folder inside /docs and link to files/production-resume.pdf, etc.
5. For reel embeds, replace placeholder links with actual YouTube/Vimeo embeds.

Design direction:
- Minimal cream, gold, deep green, black palette
- Editorial / cinematic feel
- Multi-disciplinary artist portfolio, not just an acting site
